SeblakJeletotApp

Project Android sederhana untuk aplikasi pemesanan Seblak Jeletot berbasis mobile khusus pelanggan.

Struktur utama:
- app/src/main/AndroidManifest.xml
- app/src/main/java/com/seblakjeletot/app/activity
- app/src/main/java/com/seblakjeletot/app/model
- app/src/main/java/com/seblakjeletot/app/adapter
- app/src/main/java/com/seblakjeletot/app/database
- app/src/main/res/layout
- app/src/main/res/values

Cara buka:
1. Extract ZIP.
2. Buka Android Studio.
3. Pilih Open.
4. Pilih folder SeblakJeletotApp.
5. Tunggu Gradle Sync selesai.
6. Jalankan aplikasi di emulator atau HP Android.

Catatan:
- Package: com.seblakjeletot.app
- Database lokal: SQLite
- Data menu awal otomatis dibuat lewat DatabaseHelper.
- Riwayat pesanan tersimpan di SQLite.
